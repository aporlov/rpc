# pgrpc-sql upstream

Проект предназначен для использования в качестве базы для sql-приложений, использующих pgrpc.

## Зависимости

* linux (в т.ч. bash, awk)
* docker
* make

* pgrpc

## Установка

* создать репозиторий
* настроить upstream
* выгрузить файлы проекта
* добавить свою математику


## Обновления

* обновить upstream
* обновить файлы своего проекта из текущего
